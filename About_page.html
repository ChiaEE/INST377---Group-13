This is a website that's designed to help movie directors in making film plots. This website gives users the ability to search the database by book name, author, publication date and genre. This website uses Project Gutenberg's Gutendex API to perform searches and extract information on books.

How to Navigate this website?

This website has a couple features on it that aid in navigation - The menu up top, and the homepage's search button.
