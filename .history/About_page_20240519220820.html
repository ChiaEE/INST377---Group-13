This is a website that's designed to help movie directors in making film plots. This website gives users the ability to search the database by the title of the book and ensures that the books are public Domain by allowing users to filter books based on copyright claims. This website uses Project Gutenberg's Gutendex API to perform searches and extract information on books.

How to Navigate this website?

This website has a couple features on it that aid in navigation - The menu up top, and the homepage's search button.
